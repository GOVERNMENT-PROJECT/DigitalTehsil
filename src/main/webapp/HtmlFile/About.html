WHAT IS DIGITAL TEHSIL

In this project we have developed a Digital Tehsil application which is very easy to use. It can be used with computer web browser as well as with mobile devices. Many websites are developed to reduce their use of Pen and paper but they are not fully online supportive.

This project "E-Tehsil : A Cross Terminal Web System Based on JSON and Rest" aims at making all facilities easier to access. Common user can fill up the forms and it is submitted at central database for approval. The administrator can verify all the forms which is finally stored at server.
